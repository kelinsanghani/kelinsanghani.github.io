Diabetes Prediction Using Machine Learning

This project aimed to predict diabetes occurrence using health data from the BRFSS2015 dataset. Multiple ML models—Logistic Regression, KNN, Gradient Boosting, XGBoost, and LightGBM—were trained and evaluated.

Key Findings: XGBoost achieved 90% accuracy. SHAP and LIME visualizations highlighted BMI, age, and glucose level as top predictors. These insights can support preventive healthcare decisions.
